Features
Built for Scale & Reliability

Designed for large-scale processing on cloud GPU infrastructure with minimal manual intervention.

Checkpoint Recovery

Each step maintains independent checkpoints. On failure, resume from the last saved position automatically.

Multi-GPU Parallel

Set NUM_GPUS=N to dispatch each year to a separate GPU. Linear speedup.

Optional Reranking

Enable BGE-Reranker-v2-M3 cross-encoder to refine top-k candidates for higher-precision matching.

Score Threshold

Configurable min_embed_score filters out low-confidence matches, ensuring data quality.

Row Validation

Automatic row-count checks between consecutive steps detect silent data loss immediately.

Persistent Logging

All steps write structured logs to disk for post-hoc diagnostics.
